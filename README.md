# NGO Management System

একটি সম্পূর্ণ NGO ম্যানেজমেন্ট সিস্টেম যেখানে লোন, সেভিংস, কালেকশন ম্যানেজ করা যায়।

## Features
- কাস্টমার ম্যানেজমেন্ট
- লোন ম্যানেজমেন্ট
- সেভিংস ম্যানেজমেন্ট
- দৈনিক/মাসিক রিপোর্ট
- ক্যাশ ব্যালেন্স ট্র্যাকিং

## Deploy করার নিয়ম

### Render.com এ Deploy (ফ্রি)

1. GitHub এ কোড আপলোড করো
2. Render.com এ যাও এবং সাইন আপ করো
3. "New +" > "Blueprint" ক্লিক করো
4. তোমার GitHub রিপো সিলেক্ট করো
5. Deploy হয়ে যাবে!

### Local এ Run করতে

```bash
pip install -r requirements.txt
python run.py
```
